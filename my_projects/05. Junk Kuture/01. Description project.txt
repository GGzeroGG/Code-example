It's a mobile contest app / social network where people compete by making clothes out of junk
Overall this project didn't have much in the way of direct new stuff, but it did give me a lot of small experiences

In this project I:

    Got acquainted with the auto documentation of the swagger
    I wrote a little class for firebase push notification
    Started to get acquainted with django orm
    Redefined my first manager class for the user model
    Wrote my first console command
    Getting to know data migrations
    Got to know twilio for sending sms
